CAPSET API Document

# APIについて

CAPSET の内部APIです。

APIの内容は予告なく変更されます。ご了承ください。

# APIの仕様

/all

GET: 全ての投稿を返します。

/all/:id

GET: idで絞り込んだ特定の投稿を返します。

/post

POST: 投稿を作成します。

クエリ
  • username
  • email
  • title
  • detail

username: ユーザーの名前

email: ユーザーのメールアドレス

title: ライブ配信のタイトル

detail: ライブ配信の詳細情報

/delete/:streamingkey

GET/POST: streamingkeyを指定して投稿を削除します。
